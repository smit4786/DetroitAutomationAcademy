## üìê DESIGN SPECIFICATIONS

### Color Palette (Option A - Tech Forward)
**Use EXACTLY these hex codes:**

- **Primary Brand Color:** Electric Blue
  - HEX: #0066CC
  - RGB: 0, 102, 204
  - Pantone: 300 C
  - **Primary use:** Detroit skyline/main element

- **Secondary Color (Accent):** Rust / Detroit Burn
  - HEX: #CC5522
  - RGB: 204, 85, 34
  - Pantone: 1535 C
  - **Use:** Accent lines, circuit pathways (represents Detroit industrial heritage)

- **Supporting Color:** Lime / Forward Motion
  - HEX: #66CC00
  - RGB: 102, 204, 0
  - Pantone: 375 C
  - **Use:** Growth indicators, connecting nodes (optional accent)

- **Neutrals:**
  - Black: #111111
  - Medium Gray: #666666
  - Light Gray: #F5F5F5
  - White: #FFFFFF

**Accessibility Verified:**
- Blue + white: 8.6:1 contrast ‚úÖ (AA+)
- Blue + black: 8.3:1 contrast ‚úÖ (AA+)
- Rust + white: 4.8:1 contrast ‚úÖ (AA+)

### Geometric Style

**Recommended Approach:** Low-Poly or Geometric Abstraction
- Similar to trendy "low-poly" city illustrations (modern aesthetic)
- Use clean angular lines instead of curves for skyline
- Simplify Detroit skyline to 5-8 iconic building silhouettes
- Triangular or angular geometry (suggests precision, tech)
- Scalable without losing detail (works at any size)

**Alternative Approaches:**
- Line-based (minimal, elegant, circuits as connecting lines)
- Semi-geometric (realistic building shapes + circuit overlays)
- Pure circuit abstraction (skyline outline formed by circuit paths)

**Your choice:** Pick the approach that feels right for brand positioning

### Stroke & Line Specifications

**Primary Strokes (Skyline outline):**
- Width: 2-3px (specify intentionally)
- Style: Angular/geometric (suggests precision)
- Consistency: All primary strokes EXACTLY same width

**Circuit/Secondary Lines:**
- Width: 1-1.5px (thinner than skyline for hierarchy)
- Style: Organic curves (suggests connections, technology)
- Consistency: All secondary strokes EXACTLY same width
- Connect: Network nodes or skyline points

**Color Application:**
- Skyline: Primary blue (#0066CC)
- Circuit lines: Secondary rust (#CC5522) or accent lime (#66CC00)
- Background: White or transparent
